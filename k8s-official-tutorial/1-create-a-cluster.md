## Using minikube to create a cluster

`minikube version`  

\# Running k8s cluster
\# Minikube starts a VM, and k8s cluster runs on that VM.  
`minikube start`  

\# Check kubectl  
`kubectl version`  

\# Check cluster status - master: It provides URL for cluster monitoring UI.  
`kubectl cluster-info`  

\# Check node status  
`kubectl get nodes`
